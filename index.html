Atenção

Este projeto consiste em uma simulação que exemplifica a comunicação entre o front-end e o back-end. É importante ressaltar que os dados inseridos por qualquer usuário não serão armazenados em nenhum banco de dados permanente. Portanto, ao recarregar a página, os dados anteriormente inseridos não estarão mais disponíveis. Para emular a funcionalidade de posts, utilizamos a API do JSONPlaceholder, a qual simula dados fictícios de maneira eficaz.
